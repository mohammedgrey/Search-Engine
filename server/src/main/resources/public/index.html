Welcome to the "look me up" search Engine API

Base Url: https://lookme-up.herokuapp.com

Endpoints

  • /search

    • To search for a specific sentence

    GET

    Query Parameters:

    • q: the search query string (must be encoded while sending the request)
    • page: page number (for pagination.) The default is 1
    • limit: The maximum number of search results in each page(for pagination.) The default is 10

    Full url example:

    https://lookme-up.herokuapp.com/search?q=anime%world&page=1&limit=10

    Sample response:

    [

    {"url":"https://merchdope.com/best-anime/",

    "title":"29 Best Anime Series of All Time - 2021 -MerchDope",

    "snippet":"",

    "website":"MerchDope",

    "keywords":["anim"]}

    ]

  • /history

    • To get the history of all users as suggestions as you type your search query. Returns 10 suggestions at most.

    GET

    Query Parameters:

    • q: the search query string (must be encoded while sending the request)

    Full url example:

    https://lookme-up.herokuapp.com/history?q=how%20to

    Sample response:

    [

    {"sentence":"How to code"},

    {"sentence":"How to die"},

    {"sentence":"How to disappear"},

    {"sentence":"How to fly"},

    {"sentence":"How to play"},

    {"sentence":"How to sleep"},

    {"sentence":"How to study"}

    ]
